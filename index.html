Thực hiện 1 lần duy nhất để Excel hoạt động chuẩn xác 100%.

1. Cài đặt Region (Control Panel)

1. Windows → Tìm Control Panel → Mở lên.

2. Chọn Clock and Region → Region.

3. Tab Format: Chọn Vietnamese (Vietnam) để chuẩn ngày giờ (dd/mm/yyyy).

2. Cài đặt Excel (Dấu chấm phẩy)

1. Mở Excel → File → Options → Advanced.

2. Bỏ chọn "Use system separators".
Decimal separator: để là . (chấm)
Thousands separator: để là , (phẩy)
(Trên chấm dưới phẩy)

3. Bật Macro & Tắt Protect (Bắt buộc)

1. Vào File → Options → Trust Center → Trust Center Settings...

2. Tab Macro Settings: Chọn dòng cuối Enable VBA macros... (và tích chọn "Trust access to the VBA project model" nếu cần).

3. Tab Protected View: Bỏ chọn (Uncheck) tất cả 3 ô để tắt chế độ bảo vệ khi tải file từ mạng về (tránh lỗi file bị khóa).
